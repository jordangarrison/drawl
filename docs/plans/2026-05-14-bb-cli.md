# Babashka CLI — implementation plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `drawl compile|lint|watch` as a Babashka-based CLI distributable via `bbin`, matching SPEC §5.2.

**Architecture:** A thin `cli.main` namespace under `src/cli/main.clj` wraps the existing `drawl.compiler` public API. `babashka.cli` parses options. The compiler core is `.cljc`, so bb loads it from the same `src/` paths used by JVM and CLJS — no code duplication. `watch` polls `babashka.fs/last-modified-time` (500 ms) because `babashka.fs/watch` is not present in the pinned bb 1.12 build. The SPA already defaults to `:dot`; the CLI matches.

**Tech Stack:**
- Babashka 1.12 (built-in `babashka.cli`, `babashka.fs`, `clojure.test`)
- `org.clojure/data.json` 2.4.0 (declared in `bb.edn` so bb can load `drawl.emit.excalidraw`, which `drawl.compiler` requires). Version pinned to 2.4.0 because 2.5.0 uses `definterface`, which babashka's sci runtime does not support.
- `org.babashka/cli` 0.8.61 (declared in `deps.edn` so the JVM-side `clojure -M:test` can resolve the namespace; on bb it's a built-in pod).
- Cognitect `test-runner` (already wired in `deps.edn :test` for JVM CLI smoke)

**Date:** 2026-05-14
**Spec:** [`SPEC.org`](../../SPEC.org) §5.2

**Branch:** `feat/bb-cli` — single feature branch, PR back to `main`.

Established conventions to follow:
- `.cljc` for code that must run on JVM + CLJS + bb (compiler core). `.clj` for JVM/bb-only (this CLI).
- Errors are `ex-info` w/ `:type` ∈ `#{:parse-error :walk-error :emit-error}`. Reuse — don't invent new error shapes.
- Conventional commits. Never force-push.

## Public surface (CLI contract)

```
drawl compile  --input FILE [--backend dot|mermaid|excalidraw] [--output FILE]
drawl lint     --input FILE
drawl watch    --input FILE [--backend dot|mermaid|excalidraw] [--output FILE]
```

- Default backend: `:dot`.
- No `--input`: stdin. (`bb -m cli.main compile -b dot < file.drawl`)
- No `--output`: stdout for `compile`/`watch`; silent on success for `lint`.
- Exit codes: `0` success, `1` compile/lint error, `2` bad args / unknown subcommand.

## Task 3: minimal `cli.main` to make the compile test pass

```clojure
(ns cli.main
  "drawl CLI entry. Dispatches subcommands to handlers that return integer
  exit codes. -main calls System/exit only when invoked as the process
  entrypoint; tests call -main directly and just consume the return value."
  (:require [babashka.cli :as cli]
            [clojure.java.io :as io]
            [clojure.string :as str]
            [drawl.compiler :as drawl]))

(def ^:private backend-aliases
  {"dot" :dot "mermaid" :mermaid "excalidraw" :excalidraw})

(def ^:private cli-spec
  {:input   {:alias :i :desc "Path to .drawl source (omit for stdin)"}
   :output  {:alias :o :desc "Path to write output (omit for stdout)"}
   :backend {:alias :b :desc "dot|mermaid|excalidraw" :default "dot"
             :coerce :string}})

(defn- read-source [^String input]
  (if (and input (not= "-" input))
    (slurp input)
    (slurp *in*)))

(defn- write-output [^String output ^String s]
  (if (and output (not= "-" output))
    (spit output s)
    (do (.write *out* s) (.flush *out*))))

(defn- parse-backend [^String s]
  (or (backend-aliases s)
      (throw (ex-info (str "Unknown backend: " s)
                      {:type :cli-error :backend s}))))

(defn- print-error [^Throwable t]
  (let [{:keys [type message position]} (merge {:message (ex-message t)}
                                               (ex-data t))]
    (binding [*out* *err*]
      (println (str (or type "error") ": " message
                    (when position (str " at " position)))))))

(defn compile-cmd [{:keys [input output backend]}]
  (try
    (let [src (read-source input)
          out (drawl/compile src (parse-backend backend))]
      (write-output output out)
      0)
    (catch clojure.lang.ExceptionInfo e (print-error e) 1)))

(defn lint-cmd [{:keys [input]}]
  (let [src    (read-source input)
        errors (drawl/validate src)]
    (if (nil? errors)
      0
      (do (doseq [e errors]
            (binding [*out* *err*]
              (println (str (:type e) ": " (:message e)
                            (when-let [p (:position e)] (str " at " p))))))
          1))))

(defn watch-cmd [_opts]
  ;; Placeholder — wired in Task 6.
  (binding [*out* *err*] (println "watch: not yet implemented"))
  2)

(def ^:private usage
  "Usage: drawl <subcommand> [options]

  compile  -i FILE [-b BACKEND] [-o FILE]   Emit backend output (default backend: dot)
  lint     -i FILE                          Validate without emitting
  watch    -i FILE [-b BACKEND] [-o FILE]   Recompile on file change

  Backends: dot (default), mermaid, excalidraw")

(defn -main [& args]
  (let [[sub & rest-args] args]
    (case sub
      "compile" (compile-cmd (cli/parse-opts rest-args {:spec cli-spec}))
      "lint"    (lint-cmd    (cli/parse-opts rest-args {:spec cli-spec}))
      "watch"   (watch-cmd   (cli/parse-opts rest-args {:spec cli-spec}))
      (do (binding [*out* *err*] (println usage))
          2))))
```

### 6b: Implement `tick-once` + the watch loop

- [ ] **Step 3: Replace `watch-cmd` and add `tick-once`** in `src/cli/main.clj`

Insert above `watch-cmd`:

```clojure
(defn- mtime [^String path]
  (let [f (io/file path)]
    (when (.exists f) (.lastModified f))))

(defn- tick-once
  "One iteration of the watch loop. Pure-ish: reads the file when its mtime
  has changed, writes the output, returns the next state. Returns a map
  with :status ∈ #{\"ok\" \"idle\" \"error\"} and :mtime."
  [{:keys [input output backend] :as opts} {:keys [last-mtime]}]
  (let [now (mtime input)]
    (cond
      (nil? now)
      (do (binding [*out* *err*] (println "watch: input missing:" input))
          {:status "error" :mtime nil})

      (= now last-mtime)
      {:status "idle" :mtime now}

      :else
      (let [code (compile-cmd opts)]
        {:status (if (zero? code) "ok" "error") :mtime now}))))
```

Replace the existing `watch-cmd`:

```clojure
(defn watch-cmd [{:keys [input] :as opts}]
  (when (nil? input)
    (binding [*out* *err*] (println "watch: --input is required"))
    (System/exit 2))
  (binding [*out* *err*] (println (str "watch: " input " (Ctrl-C to stop)")))
  (loop [state {:last-mtime nil}]
    (let [next-state (tick-once opts state)]
      (when (= "ok" (:status next-state))
        (binding [*out* *err*] (println (str "watch: recompiled @ " (:mtime next-state)))))
      (Thread/sleep 500)
      (recur (assoc state :last-mtime (:mtime next-state))))))
```

(ns cli.smoke-test
  "Spawns `bb -m cli.main ...` against every fixture in examples/.
  Skipped automatically when bb is not on PATH."
  (:require [clojure.java.io :as io]
            [clojure.java.shell :as sh]
            [clojure.string :as str]
            [clojure.test :refer [deftest is testing]]))

(defn- bb-available? []
  (try (zero? (:exit (sh/sh "bb" "--version"))) (catch Exception _ false)))

(defn- example-files []
  (->> (file-seq (io/file "examples"))
       (filter #(str/ends-with? (.getName ^java.io.File %) ".drawl"))
       sort))

(deftest bb-cli-compile-every-example-to-dot
  (if-not (bb-available?)
    (println "skipping bb smoke test: bb not on PATH")
    (doseq [^java.io.File f (example-files)]
      (testing (.getName f)
        (let [{:keys [exit out err]}
              (sh/sh "bb" "-m" "cli.main" "compile" "-i" (.getAbsolutePath f) "-b" "dot")]
          (is (zero? exit) (str "non-zero exit for " (.getName f) ": " err))
          (is (str/starts-with? out "digraph G {")
              (str "unexpected dot output for " (.getName f))))))))

(deftest bb-cli-lint-every-example
  (if-not (bb-available?)
    (println "skipping bb smoke test: bb not on PATH")
    (doseq [^java.io.File f (example-files)]
      (testing (.getName f)
        (let [{:keys [exit err]} (sh/sh "bb" "-m" "cli.main" "lint" "-i" (.getAbsolutePath f))]
          (is (zero? exit) (str "lint failed for " (.getName f) ": " err)))))))
